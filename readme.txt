Tool to resize the image appropriately

== Description ==

You will be able to resize smartphone, tablet PC for the image in the article. This Plugin is the ideal tool for Responsive Web design.

== Installation ==

1. Upload the entire `auto-image-resize` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Plug-in Usage ==

In order to use this plug-in, there is no special knowledge.
enable the plug-in, will start running.

If necessary, from the configuration screen, you can also specify the width of each device.
